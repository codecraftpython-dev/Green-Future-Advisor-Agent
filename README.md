# Green Future Advisor Agent 🌿

AI-Powered Sustainable Living Guidance Assistant

## Overview
Green Future Advisor Agent is an AI-based sustainability assistant that helps users make environmentally responsible decisions through personalized recommendations.

## Problem
Many people want to live sustainably but lack personalized guidance about reducing their environmental impact.

## Solution
An AI agent that provides guidance on:
- Sustainable lifestyle choices
- Carbon footprint awareness
- Energy conservation
- Waste reduction
- Climate-friendly actions

## AI Technologies Used
- IBM watsonx Orchestrate
- IBM watsonx.ai
- IBM Granite Models
- Retrieval-Augmented Generation (RAG)
- Agentic AI Workflow
- Prompt Engineering
- Conversational AI
